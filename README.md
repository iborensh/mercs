### How to start

**Note** that this seed project requires **node >=v6.9.0 and npm >=3**.

In order to start the project use:

```bash
$ git clone https://github.com/iborensh/mercs.git
$ cd DB
$ python db.py
$ cd ../client
$ npm install
$ npm start
```
